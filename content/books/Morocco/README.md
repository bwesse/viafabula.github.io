# Morocco Book - VIA Fabula Language Learning Platform

This book contains travel stories about various cities and locations in Morocco, designed for language learning at multiple CEFR levels.

## Book Structure

### Chapters (12 total)
1. **00_intro** - Introduction to Morocco
2. **01_Marrakesh** - The vibrant heart of Morocco
3. **02_Fez** - Morocco's cultural and spiritual capital
4. **03_Casablanca** - Morocco's cosmopolitan economic hub
5. **04_Rabat** - Morocco's elegant capital
6. **05_Tangier** - The gateway between Africa and Europe
7. **06_Chefchaouen** - The enchanting blue pearl
8. **07_Agadir** - Morocco's premier beach resort
9. **08_Merzouga** - Gateway to the Sahara desert
10. **09_Essaouira** - The charming coastal town
11. **10_Meknes** - The imperial city
12. **11_Ouarzazate** - The Hollywood of Morocco

### Languages (4 total)
- English
- Spanish (Español)
- German (Deutsch)
- French (Français)

### CEFR Levels (9 total)
- **a0** - Beginner (~100 words) - Very simple sentences, present tense
- **a1** - Elementary (~500 words) - Simple sentences, basic past tense
- **a2** - Pre-Intermediate (~1000 words) - More complex sentences, questions
- **b1** - Intermediate (~2000 words) - Complex sentences, various tenses
- **b2** - Upper-Intermediate (~4000 words) - Idiomatic expressions, nuanced language
- **c1** - Advanced (~8000 words) - Abstract topics, sophisticated language
- **c2** - Proficiency (~16000 words) - Near-native complexity
- **native** - Native speaker level - Modern, natural-sounding text
- **original** - Original version (same as native for this book)

## Content Format

### Story Files (.md)
Each .md file contains a travel narrative appropriate for the CEFR level, telling about visiting that particular Moroccan city.

### Question Files (_q.json)
Each _q.json file contains 3 comprehension questions with the following structure:
```json
{
  "questions": [
    {
      "id": "q1",
      "question": "Question text?",
      "options": [
        "Option 1",
        "Option 2",
        "Option 3"
      ],
      "correctIndex": 0
    }
  ]
}
```
