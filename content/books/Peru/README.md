### CEFR Levels (9 total)
- **a0** - Beginner (~100 words) - Very simple sentences, present tense
- **a1** - Elementary (~500 words) - Simple sentences, basic past tense
- **a2** - Pre-Intermediate (~1000 words) - More complex sentences, questions
- **b1** - Intermediate (~2000 words) - Complex sentences, various tenses
- **b2** - Upper-Intermediate (~4000 words) - Idiomatic expressions, nuanced language
- **c1** - Advanced (~8000 words) - Abstract topics, sophisticated language
- **c2** - Proficiency (~16000 words) - Near-native complexity
- **native** - Native speaker level - Modern, natural-sounding text
- **original** - Original version (same as native for this book)

## Content Format

### Story Files (.md)
Each .md file contains a travel narrative appropriate for the CEFR level, telling about visiting that particular Peruvian location.

### Question Files (_q.json)
Each _q.json file contains comprehension questions with the following structure:
```json
{
  "questions": [
    {
      "id": "q1",
      "question": "Question text?",
      "options": [
        "Option 1",
        "Option 2",
        "Option 3"
      ],
      "correctIndex": 0
    }
  ]
}
```
