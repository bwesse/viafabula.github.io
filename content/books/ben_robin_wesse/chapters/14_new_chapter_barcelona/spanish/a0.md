# Un Nuevo Capítulo: Barcelona

Colombia fue divertido. Escribimos nuestras tesis allí.

Ahora estoy en Barcelona. Es una nueva ciudad. Es una nueva vida.

Estudio en la UPC. Es un programa de máster. Aprendo sobre robots.

Barcelona es hermosa. Hay playas. Hay calles antiguas.

Mi español es bueno pero no excelente. Todavía aprendo cada día.

Quería aprender mejor. Hice una página web. Se llama VIA Fabula.

VIA Fabula me ayuda a leer historias. Puedo leerlas en diferentes niveles. Puedo cambiar entre español, inglés y alemán.

Escribí este libro para la página web. Es mi historia.

El estudiante de arte hizo una aplicación web. Eso es increíble.

Pienso en mi viaje. Hamburgo. Canadá. Perú. Klagenfurt. Ahora Barcelona.

Era estudiante de arte. Ahora construyo drones. Ahora estudio más. Ahora hago páginas web.

No sé todo. Eso está bien. El camino se muestra solo.

Hay montañas cerca. Quiero hacer snowboard de nuevo. Pero primero, termino este capítulo. En español.
